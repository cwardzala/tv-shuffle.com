# TV Shuffle

Marketing website for the TV Shuffle iOS app.

## Logo Colors

The brand colors are extracted from the app logo (`img/tvs-logo-960.png`):

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| White | `#f5f5f5` | Primary text |
| Yellow | `#faf354` | Accent/gradient |
| Green | `#8bf75a` | Accent/gradient |
| Cyan | `#5cf8f8` | Accent/gradient |
| Pink | `#f89acc` | Accent/gradient |
| Red | `#dd576b` | Accent/gradient |
| Blue | `#5b82f5` | Primary accent |

## Development

```bash
npm run dev    # Start dev server
npm run build  # Build for production
```

## Tech Stack

- 11ty (Eleventy) static site generator
- Bootstrap 5 (CSS only)
- Bootstrap Icons
